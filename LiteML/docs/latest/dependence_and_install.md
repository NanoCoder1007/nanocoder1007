## 依赖与安装

### 运行时依赖

| Crate    | 版本 | 用途                         |
| :------- | :--- | :--------------------------- |
| `anyhow` | ^1.0 | 错误处理（统一 `Result<T>`） |
| `clap`   | ^4.0 | 命令行参数解析               |
| `regex`  | ^1.0 | 词法分析的正则表达式引擎     |

> 注意：本项目使用 Rust 1.71+（2024‑stable）编译。

### 安装

```bash
cargo install liteml
```

验证安装
```bash
liteml
```
