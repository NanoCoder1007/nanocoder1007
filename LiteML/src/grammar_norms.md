# 语法规范
LiteML 采用类声明式语法，结构清晰、易于书写与解析。

## 3.1 根结构
所有内容必须包裹在 `page {}` 根块内：
```
page {

}
```

## 3.2 标题元素
使用 `title()` 声明标题，支持 `level` 参数控制标题层级：
```
title("文本内容", level=1);
```
- 文本必须使用双引号包裹
- `level` 取值范围：1~6，默认值为 1
- 语句末尾必须加分号

## 3.3 完整示例
```
page {
    title("LiteML 编译器", level=1);
    title("快速开始", level=2);
    title("安装与构建", level=3);
    title("基础语法", level=3);
}
```
