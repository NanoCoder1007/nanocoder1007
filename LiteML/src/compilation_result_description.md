# 编译结果说明
LiteML 会生成**标准、可直接部署**的 HTML5 文档：
- 自动设置 `UTF-8` 编码
- 页面标题取自第一个 `title` 文本
- 标题按 `h1~h6` 规范渲染
- 格式缩进统一、结构整洁

示例输出：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>LiteML 编译器</title>
</head>
<body>
    <h1>LiteML 编译器</h1>
    <h2>快速开始</h2>
</body>
</html>
```
